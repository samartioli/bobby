# Basic Web Page

## Check out the project from git
(This only has to be done once)

    cd ~/Documents
    git clone https://github.com/samartioli/bobby.git
    cd bobby

## Start the server

    python -m SimpleHTTPServer

## Test out the page

In your web browser, go to [http://localhost:8000](http://localhost:8000)
